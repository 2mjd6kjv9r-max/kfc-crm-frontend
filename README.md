# KFC CRM / CDP & Admin Module

This is a production-ready CRM system designed for KFC, featuring:
- Real-time Dashboard
- Funnel Analytics
- Dynamic Segments
- Customer Lifecycle Management
- Retention & Churn Analysis
- Cohort Analysis
- Automation Builder (IF/THEN)
- Data Explorer

## Tech Stack
- Client: React (Vite)
- Server: Node.js (Express)
- Database: SQLite (for portability) / Postgres (Production compatible)
- Styling: Vanilla CSS

## Setup
1. `cd client` && `npm install`
2. `cd server` && `npm install`
3. Start Client: `cd client` && `npm run dev`
4. Start Server: `cd server` && `npm run start`
